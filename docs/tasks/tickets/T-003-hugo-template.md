# T-003: 記事JSON → Hugo ページ変換テンプレート作成

## 概要
`content/posts/YYYY-MM-DD.json` に格納された記事データを Hugo で HTML ページとしてレンダリングするためのテンプレートを作成する。

## やること
- JSON データを読み込む Hugo テンプレート（`layouts/`）の作成
- 日別記事一覧ページのレイアウト設計
- 各記事のタイトル、要約、タグ、スコア、ソース、元記事リンクの表示
- タグによるフィルタリングやソート機能（任意）

## 完了条件
- サンプルの JSON ファイルを配置して `hugo build` が成功する
- 日別ページで記事一覧が正しく表示される
- 元記事へのリンクが機能する
