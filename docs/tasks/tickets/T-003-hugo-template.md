# T-003: 記事JSON → Hugo ページ変換テンプレート作成

**ステータス**: ✅ 完了（2026-03-15）

## 概要
記事データを Hugo で HTML ページとしてレンダリングするためのテンプレートを作成する。

## やること
- 記事データを読み込む Hugo テンプレート（`layouts/`）の作成
- 日別記事一覧ページのレイアウト設計
- 各記事のタイトル、要約、タグ、スコア、ソース、元記事リンクの表示
- タグによるフィルタリングやソート機能（任意）

## 完了条件
- サンプルデータを配置して `hugo build` が成功する
- 日別ページで記事一覧が正しく表示される
- 元記事へのリンクが機能する

## 完了メモ
- **設計変更**: コンテンツ形式を当初の `YYYY-MM-DD.json` から `YYYY-MM-DD.md`（YAML フロントマター）に変更。Hugo が JSON ファイルをネイティブコンテンツとして認識しないため、Hugo 標準の Markdown + YAML フロントマター形式を採用
- `layouts/posts/single.html` にカード形式の記事テンプレートを作成
- 各記事にソースバッジ、スコア、タグ、元記事リンク、日本語要約を表示
- サンプルデータ（3件）で `hugo build` 成功確認済み
- PaperMod テーマの CSS 変数（`var(--border)` 等）を活用してテーマ互換性を確保
