# T-002: Hugo サイト初期構築

**ステータス**: ✅ 完了（2026-03-15）

## 概要
Hugo の初期設定を行い、テーマを選定・導入してサイトの骨格を作る。

## やること
- `hugo new site` でプロジェクトを初期化（既存リポジトリに統合）
- `hugo.toml` の基本設定（サイトタイトル、言語、ベースURL等）
- テーマの選定と導入（Git submodule で管理）
- トップページのレイアウト作成

## 完了条件
- `hugo server` でローカルプレビューが動作する
- テーマが適用されてスタイルが反映されている
- トップページに記事一覧が表示される構成になっている

## 完了メモ
- `hugo new site . --force` で既存リポジトリに統合
- テーマは **PaperMod** を選定（軽量・レスポンシブ・ダークモード対応・拡張性あり）
- Git submodule (`themes/PaperMod`) で管理
- `hugo.toml` に日本語ロケール、サイトタイトル「Tech News Daily」を設定
- `hugo server` でローカルプレビュー動作確認済み
