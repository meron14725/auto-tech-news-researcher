# T-021: スキル評価フレームワーク

## 概要
OpenClaw の research-trend-news スキルの品質を検証するためのテストケースと検証スクリプトを作成する。

## 依存
- T-014.5（スキル変換）

## やること
- テストケースをデータファイルとして定義（`tests/eval/`）
- 検証スクリプト `scripts/validate-skill.sh` を作成:
  - 出力フォーマット準拠チェック（YAML front matter 構造）
  - スコアリングロジック検証（既知の記事に対する期待スコア）
  - 日本語要約品質ヒューリスティクス（文字数、文字種）
- フィードバック改善ループ（T-026）との統合

## 完了条件
- テストケースが存在する
- 検証スクリプトが現在のスキル出力でパスする
- 改善ループから呼び出し可能
