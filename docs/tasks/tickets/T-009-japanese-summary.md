# T-009: 日本語要約生成機能

## 概要
採用された各記事について、日本語タイトルと3〜5文の日本語要約を Claude に生成させる。

## やること
- 元記事のタイトルを自然な日本語に翻訳
- 記事内容（タイトル・概要・URL）をもとに3〜5文の日本語要約を生成
- 適切なタグ（例: AI, LLM, OSS, Web, インフラ）を付与
- 英語記事・日本語記事両方に対応

## 完了条件
- 各記事に `title`（日本語）と `summary`（日本語3〜5文）が生成される
- `original_title` に元のタイトルが保持される
- `tags` が適切に付与される
