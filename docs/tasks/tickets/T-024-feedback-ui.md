# T-024: 記事カードにフィードバック UI 追加

## 概要
Hugo テンプレートの各記事カードに👍👎ボタンを追加し、フィードバック API と連携する。

## やること
- `layouts/posts/single.html` に👍👎ボタンを追加
- JavaScript で `/api/feedback` に POST
- localStorage で投票状態を記憶（ページリロードでも状態維持）
- PaperMod の CSS 変数に合わせたスタイリング
- エラー時は何も表示しない（UX を壊さない）

## 依存
- T-023（フィードバック API）

## 完了条件
- 各記事カードに👍👎ボタンが表示される
- クリックで API に送信され、ボタンがハイライトされる
- 投票状態がリロード後も維持される
