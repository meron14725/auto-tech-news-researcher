# T-014.5: OpenClaw スキル変換

## 概要
現在の `.claude/skills/research-trend-news/SKILL.md` を OpenClaw のスキル形式に変換する。

## 依存
- T-013.5（OpenClaw Gateway 設定）

## やること

### 1. スキルファイルの形式変換
OpenClaw 形式の YAML frontmatter を追加:
```yaml
---
name: research-trend-news
description: 技術トレンドニュースを複数ソースから収集し、日本語で要約して記事ファイルを生成する
version: 1.0.0
metadata:
  openclaw:
    requires:
      bins: [curl, jq, git]
---
```

### 2. ツール名の変更
| Claude Code | OpenClaw |
|-------------|----------|
| Bash | bash |
| Read | read |
| Write | write |

### 3. 処理フロー（流用）
以下は既存 SKILL.md からほぼそのまま移植:
- 前日の日付取得
- HN / Zenn / dev.to からの記事取得
- interest_score >= 7 のフィルタリング
- 日本語要約生成
- `content/posts/YYYY-MM-DD.md` 出力
- `data/processed_urls.json` 更新

### 4. スキルインストール
```bash
openclaw skills add ./path/to/research-trend-news
```

### 5. 手動テスト
スキルを手動実行し、正しい形式の Markdown ファイルが生成されることを確認。
