# T-010: 重複排除機能

**ステータス**: ✅ 完了（2026-03-15）

## 概要
`data/processed_urls.json` で処理済みURLを管理し、同じ記事が複数回処理されることを防ぐ。

## やること
- `processed_urls.json` の読み込み・書き込みロジック
- 記事取得時に URL を照合して処理済みをスキップ
- 新規処理した記事の URL を `processed_urls.json` に追記
- ファイルが肥大化した場合の対策検討（将来的に古いURLを削除するなど）

## 完了条件
- 同じ記事が2日連続で処理されない
- `processed_urls.json` が正しく更新される
- ファイルが存在しない場合でもエラーにならない（初回実行対応）

## 完了メモ
- SKILL.md に重複排除フローを定義（処理前に processed_urls.json を読み込み、URL 照合で除外）
- 処理後に新規 URL を追加して上書き保存
- リストが1000件を超えた場合は古いものから削除するルールを設定
- 初期ファイル `[]` は T-001 で作成済み
